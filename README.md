SoundTap is a lightweight application that captures audio from your computer and saves it as a WAV file.


* Record desktop system audio or whatever is playing from your speakers/headphones
* Automatically save recordings to a default or chosen folder
* Saves audio as a .WAV file with a timestamp
* Clean UI
* Packaged as a standalone Windows app
* Can be installed (SoundTap 1.0 Installer)
* Or simply opened as a portable .exe (SoundTap.exe)
